# Matrix-Theme
 My first attempt to create theme for Backets
 Its to give you the feel of hacker terminal from movies like Matrix. I hope you like it!

## How to install (only Brackets 1.0 and later)
File >> Extension manager.<br/>
Click the tab called "Themes".<br/>
Type "Matrix Theme" in the top right search bar.<br/>
Click "Install".<br/>
Click Close on the pop up.<br/>
Click Close on the extension manager.<br/>
View >> Themes.<br/>
Current Theme change to the theme you installed.<br/>
Click Done.<br/>
